# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.x     | Yes    |

## Reporting a Vulnerability

**Please do not report security vulnerabilities via public GitHub Issues.**

Email: security@linkgenetic.com
Subject: [SECURITY] Brief description

### What to include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if available)

### What to expect

- Acknowledgement within 48 hours
- Status update within 7 days
- Fix or mitigation within 90 days for critical issues

We follow coordinated disclosure.

## Scope

This policy covers:
- Client SDK libraries (sdk/js, sdk/python, sdk/java)
- Specification documents (spec/, draft/)

For resolver infrastructure security issues, contact licensing@linkgenetic.com.
